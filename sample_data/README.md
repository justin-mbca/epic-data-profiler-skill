# Sample Data Files

This folder contains synthetic Epic-style data files for testing the Epic Data Profiler skill.

- `sample_epic.csv`, `sample_epic.json`, `sample_epic.parquet`: Small test files (4 rows)
- `sample_epic_large.csv`, `sample_epic_large.json`, `sample_epic_large.parquet`: Large test files (1000 rows)
- `create_large_samples.py`, `create_parquet_sample.py`: Scripts to regenerate these files if needed

All data is synthetic and contains missing values and realistic distributions for robust testing.
